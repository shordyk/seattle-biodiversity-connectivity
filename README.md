# seattle-biodiversity-connectivity
Independent spatial modeling project, where I use ArcGIS and Stanford Natural Capital's InVEST to map urban areas with high potential for pollinator and bird habitat connectivity based on potential green roofs sites and existing tree canopy. 


Data sources:
Building outlines - https://data-seattlecitygis.opendata.arcgis.com/datasets/1ea2f226716e417e8a554476f4337c51_0/explore?location=47.614504%2C-122.336836%2C11
Park boundaries - https://seattlecitygis.maps.arcgis.com/home/item.html?id=8c994f7dd0a54346b504b853679dd099
Impervious surface - Fractional Impervious Surface (CONUS) 2023, https://www.mrlc.gov/data
Tree Cover - CONUS 2023
City Limits - https://data-seattlecitygis.opendata.arcgis.com/datasets/SeattleCityGIS::seattle-city-limits-2/about
